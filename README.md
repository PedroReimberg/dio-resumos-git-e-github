## Comandos Git Bash


| Comando  | Descrição |
| ------------- | ------------- |
| cd PASTA ou cd .. | Mudar diretório  |
| git clone URL   | Clonar   |
| git clone URL --branch feature-1 --single-branch   |    |
| cat ARQUIVO   | Ler arquivo   |
| git status   |  Status do diretório  |
| git remote add origin URL   |  Adicionar URL de origem  |
